# 《模式识别基础》第 1 章绪论复习重点（纯概念梳理，适配课件考点）

## 一、核心基础概念（选择 / 填空 / 判断高频）

### 1. 模式识别定义

采用计算机 / 机器模仿人脑，对现实事物完成描述、分类、判断、识别；也叫机器识别、计算机自动识别。

### 2. 模式

1. 广义：时空内可观察、可区分异同的物体，**模式是事物承载的信息，不是事物本身**；
2. 三大特性：可观察性、可区分性、相似性；
3. 模式类：样本归属的类别。

### 3. 样本相关术语

- 样本：研究对象单个个体；样本集：多个样本集合；
- 已知样本：带类别标签；未知样本：只有特征、无类别标签；

### 4. 特征与特征空间

- 特征：表征样本的属性 / 量；
- 特征向量：多个特征组成向量 \(\boldsymbol{x}=[x_1,x_2,\dots,x_d]^T\)；
- 特征空间：特征向量张成的 d 维空间，**每个样本是空间内一个点**。

## 二、模式识别两大主流方法（简答题核心）

### 1. 基于知识的方法（定性 / 结构路线）

1. 核心思路：人工整理特征 - 类别推理规则，依靠专家知识推理；
2. 代表：专家系统、句法（结构）模式识别；
3. 适用：结构特征明显的对象（图像部件、文字语法结构）。

### 2. 基于数据的方法（定量 / 统计路线，课程重点）

1. 核心思路：用带标签训练集训练分类器，自动学习特征与类别映射；基础是**统计模式识别**；
2. 定量描述：用数值特征向量表示样本（水果重量尺寸、图像像素向量、语音采样序列）；
3. 结构描述：用部件 + 关系描述（房屋由屋顶、墙、门窗构成）；
4. 统计模式识别本质：在特征空间划分不同子空间，寻找类别分界面。

## 四、模式识别发展历史（选择题）

1. 1930s Fisher：统计分类理论，统计模式识别奠基；
2. 1929 Tauschek：数字阅读机；
3. 1950s 傅京孙：句法模式识别（国际模式识别协会首任主席，傅京孙奖为领域最高奖）；
4. 1960s Zadeh：模糊集，模糊模式识别；
5. 1980s Hopfield：神经网络模型；
6. 1990s：SVM、小样本学习兴起；
7. 2010 后：深度学习、迁移学习；近年各类大模型（DeepSeek、ChatGPT 等）属于模式识别落地系统；
8. 2024 诺奖：Hopfield、Hinton，表彰神经网络基础贡献。

## 五、典型模式识别系统（必考简答，鲑鱼鲈鱼案例 + 车牌识别）

### 1. 完整系统两大阶段

#### （1）学习 / 设计阶段（训练）

数据获取 → 预处理 → 特征提取与选择 → 分类器设计（优化求解参数）

#### （2）识别 / 推理阶段（测试）

数据获取 → 预处理 → 特征提取选择 → 分类器决策 → 输出分类结果

### 2. 各模块概念

1. **数据获取**：相机、传感器、雷达等采集原始信号 / 图像；

2. **预处理**：降噪、图像平滑、二值化、校正、信号增强，消除光照、噪声干扰；

3. **特征提取与选择**：提取区分性特征，筛选最优特征，提升类别紧致性；

4. 分类器设计

	：选定模型（线性 / 非线性），用训练数据优化参数，确定类别分界面；

	- 线性分界面：直线 \(ax+by+c=0\)；
	- 非线性分界面：曲线，区分能力更强，但易过拟合。

### 3. 鲑鱼鲈鱼分类案例核心结论

1. 单特征（长度）误差大；单特征（亮度）误差降低；双特征组合误差大幅下降；
2. 模型复杂度与泛化能力：
	- 欠拟合：模型太简单（纯线性），训练 + 测试误差都高；
	- 适中模型（二次曲线）：训练误差低、泛化能力强；
	- 过拟合：复杂非线性曲线，训练误差极低，但测试误差大，泛化差。

### 4. 实例：车牌识别系统流程

图像采集 → 车牌定位 → 倾斜校正 → 字符分割 → 字符识别。

## 六、设计模式识别系统的核心问题（简答）

1. **分类器建模**：人为选定分类函数类型（线性 / 非线性）；

2. **参数优化**：以准则函数极值为目标，用训练样本求解最优参数；不同准则得到不同分类器；

3. **特征紧致性**：同类样本在特征空间聚集、异类分隔清晰；紧致性越好，分类越简单；特征提取可优化紧致性；

4. 相似性度量

	：基于距离衡量样本相似度，统计模式识别底层基础；

	

	常用三种度量：欧氏距离、曼哈顿距离（绝对值和）、向量夹角余弦距离。
